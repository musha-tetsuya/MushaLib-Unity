# UI.ReverseMask
逆マスクを標準機能＆ノーコードで実現します。  
* https://www.zkn0hr.com/unity-reverse-mask/

## 使用方法
1. マスク側のImageのマテリアルに`ReverseMask.mat`を設定。
2. マスクされる側のImageのマテリアルに`ReverseMasked.mat`を設定。
3. 親子付けする必要は無いが、Hierarchyではマスク側のImageが上になっている必要がある。
