# UI.Layouting

## RectTransformScaler

## SafeArea
RectTransformのサイズをセーフエリア範囲に調整するコンポーネントです。

### 使用方法
1. Canvasの下にRectTransformを配置し、SafeAreaコンポーネントを付与してください。
2. SafeAreaのInspectorを設定します。
   * `RectTransform`に自身のRectTransformを設定
   * `CanvasRectTransform`にはCanvasのRectTransformを設定
